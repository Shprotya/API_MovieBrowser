# TMDB Movie Explorer

A responsive movie browsing application built with vanilla **JavaScript**, **HTML**, and **CSS**. This project utilizes **The Movie Database (TMDB) API** to fetch and display comprehensive movie data, offering a robust user experience with search, filtering, and sorting functionalities.

## Features

* **Dynamic Movie Grid:** Displays movies in a responsive, visually appealing grid.
* **Discover Categories:** Quickly browse movies by **Trending Now**, **Popular**, **Top Rated**, and **Upcoming**.
* **Search Functionality:** Search for movies by title using the TMDB `/search/movie` endpoint.
* **Genre Filtering:** Filter the movie grid by selecting a specific genre from a dropdown, populated dynamically from the TMDB API.
* **Sorting Options:** Sort the current movie list by **Popularity**, **Rating**, **Year**, or **Title (A-Z)**.
* **Detailed Movie Modal:** Click any movie card to open a modal with in-depth information, including:
    * Title and Tagline
    * Rating and Vote Count
    * Runtime
    * Release Date
    * Budget and Revenue
    * Genres and Overview

## Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **API:** [The Movie Database (TMDB) API](https://www.themoviedb.org/documentation/api)
* **Deployment:** Static hosting (e.g., GitHub Pages, Vercel, Netlify)
